# ERP for Clinic Management System
## ERP Clinic Management System

Tech Stack: ASP.NET, C#, SQL Server

## Description

Developed a full-featured Enterprise Resource Planning (ERP) system for clinic operations that centralizes patient management, appointment scheduling, and administrative workflows.

## Key Features
Patient registration and medical record management
Doctor scheduling and appointment booking system
Inventory tracking for medicines and medical supplies
Automated report generation for clinic performance and operations
Role-based access for administrative and medical staff
Technical Highlights
Implemented backend logic using ASP.NET and C#
Integrated SQL Server database for structured patient and clinic data
Designed modular architecture to manage appointments, inventory, and reporting
Built secure data management workflows for sensitive patient information
